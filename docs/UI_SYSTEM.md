# UI System

The dashboard uses a professional operations-console design system inspired by the scanned Craft template from Shadcn Studio, adapted for a dense software-company control panel instead of a marketing landing page.

Source scanned:

- https://shadcnstudio.com/preview/templates/shadcn-nextjs-craft-landing-page
- Live preview: https://shadcn-nextjs-craft-landing-page.vercel.app/

## Design Tokens

- Background: warm off-white `#f6f3ee`.
- Surface: `#fffdf8`.
- Text: near-black `#181716`.
- Brand: teal `#1d7f6e`.
- Accent: amber `#b45309`.
- Radius: 8px for cards, panels and buttons.
- Typography: system UI stack with tight operational sizing.

## Interaction

- Sidebar navigation with active state.
- Thin custom scrollbars.
- Lightweight fade/rise animations.
- Dense panels for repeat workflows.
- Lucide icon pack for all UI controls.

## Views

- Factory command center.
- Company org chart.
- Agent configuration.
- MCP catalog configuration and export.
